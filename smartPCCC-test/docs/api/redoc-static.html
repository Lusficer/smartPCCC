/camera

/camera/:camera

Process images via HTTP or MQTT for configured cameras.

cameras:
  driveway:
    snapshot:
      topic: driveway/snapshot
      url: http://192.168.1.1/latest.jpg
path Parameters
camera
required
string
Example: office
query Parameters
attempts
string
Example: attempts=2

Number of attempts before stopping without a match

break
string
Example: break=true

Break attempt loop if a match is found

Responses

http://localhost:3000/api/camera/{camera}
